Bait Ball Ecosystem

Technology: p5.js, processing

View Live Project

View Code

The goal of this project was to explore real life animal behaviors by recreating the system of a predatory fish meadering through a bait ball. Each fish in this system is an autonomous agent, aware of its surroundings and able to make decisions based on that environment. Schooling behavior was achieved using Craig Reynolds' flocking model, in which each fish has three key behaviors, separation (don't bump into other fish), alignment ( go in the same direction as the school), and cohesion ( stay close to other fish). Fear behavior takes from the seperation algorithm, just that now you are really not trying to bump into a shark! Interactive sliders were added so that a user is able to see how different weights on forces of these behaviors changes the school behavior.

From a development and design standpoint, this project reinforced the value of breaking problems down into bite sized chunks and creating dedicated test environments. Especially when dealing multiple autonomous objects it can become very difficult to track down a problem when you have a bunch of fish interacting with one another!
