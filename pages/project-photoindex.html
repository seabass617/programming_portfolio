Photography Index (Concept)

Technology: Figma, Rails API, React

Breif: The goal of this project is to create an index based photography blog for myself that focuses on simple storytelling and allows for fast management of content directly on the site.

Design Process: The inspiration for this project came from James Jean’s indexing style in his sketchbook “Process Recess”. The index is essentially a continuous wall of text, where each page of the index is noted, then followed by an anecdote about the respective sketch. I’m a photographer and I was curious as to how an indexing style like this would translate over to a digital format for showcasing my photos. Through the development of this project, I am exploring the idea of a story first approach when it comes to viewing photos, and creating something that will help me organize the moments from different chapters in my life in a more intuitive manner. I hope that by forcing users to read and develop thoughts before they see an image, that they will be able to understand me more as a human being and maybe ask themselves questions about how engaged they usually are when looking at photos of other people’s lives.

Moving forward: The plan is to build the project using a rails API for the backend, which will allow for a variety of front end applications or styles to be used by myself or others. And I plan on using React to build out the front end since I don’t know it yet and I feel as though this would be a great application for it.
